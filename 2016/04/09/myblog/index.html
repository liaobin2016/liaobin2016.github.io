//修改配置后保存
npm install hexo-deployer-git --save

hexo g
hexo d

##hexo主题更新

例如    
git clone https://github.com/hexojs/hexo-theme-light.git themes/light
修改根目录下的_config文件:
theme: light

主题下载：https://github.com/hexojs/hexo/wiki/Themes
